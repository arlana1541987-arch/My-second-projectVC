# 💰 Семейный бюджет

Простое веб-приложение для учёта семейных доходов и расходов. Работает полностью в браузере, данные сохраняются локально.

## Возможности

- Учёт доходов и расходов
- Автоматический подсчёт баланса и процента трат
- Сохранение данных в `localStorage`
- Светлая и тёмная тема
- Адаптивный дизайн для мобильных устройств

## Как запустить

### Вариант 1 — просто открыть файл

Откройте `index.html` в любом современном браузере (Chrome, Firefox, Edge).

### Вариант 2 — локальный сервер

```bash
# Python 3
python -m http.server 8080

# или Node.js
npx serve .
```

Затем откройте в браузере: `http://localhost:8080`

## Как пользоваться

1. На главной странице нажмите **«Открыть калькулятор»**
2. Выберите тип операции: доход или расход
3. Введите описание и сумму, нажмите **«Добавить»**
4. Следите за статистикой в карточках сверху
5. Для удаления операции нажмите **«Удалить»** в таблице
